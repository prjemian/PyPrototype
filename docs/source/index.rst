Overview
========

:mod:`PyPrototype` is a model layout for a Python project in a GitHub repository.
It is planned that the project will be made available through the Python
Packaging Index (PyPI) and the documentation published at ReadTheDocs.org.

To build your own Python project using this project as a template,
see the section :ref:`usage`.
